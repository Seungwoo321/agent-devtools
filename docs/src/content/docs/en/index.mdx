## What the picker ships

When you pick an element, the widget hands the agent the evidence to reason with — not a screenshot:

<div class="adt-evidence not-content">
  <img
    src="/picker-evidence.png"
    alt="The agent-devtools composer with a picked element's evidence tooltip showing its source file, component chain, and a unique CSS selector."
  />
  <ul>
    <li>
      <strong>Component chain</strong> — the named ancestors up to the root,
      e.g. <code>OrderSummary → App</code>.
    </li>
    <li>
      <strong>Source file</strong> — the authored file the element renders from,
      so the agent opens the right code first.
    </li>
    <li>
      <strong>Selector + outerHTML</strong> — the exact node, so the agent can
      still locate the markup when no source path resolves.
    </li>
  </ul>
</div>
