agent-devtools is an in-page agent devtool. A floating widget in your dev
server lets you prompt your local Claude Code in natural language; Claude edits
your source files directly while you stay in the browser.

It targets developers who already use a Claude Pro / Max subscription locally
and want the AI loop inside the browser tab they are already looking at —
without context-switching to an IDE.
